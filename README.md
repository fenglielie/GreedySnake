# GreedySnake
c++经典项目贪吃蛇游戏控制台版，源代码都是参考这个仓库
https://github.com/silence1772/GreedySnake
几乎没有改动，我只是进行了整理，并且修改文件编码为UTF-8，添加了Makefile，重新进行了编译可以正常运行。


原仓库中附带的教程地址：http://blog.csdn.net/silence1772/article/details/55005008
